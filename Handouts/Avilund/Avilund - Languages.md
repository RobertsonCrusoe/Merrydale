Languages

The rules for language use have been expanded for use in the Avilund setting.

There are four major languages in this setting, each with a number of dialects. Most characters will start off with ‘New Imperial’ as their native language, and will choose one dialect from the regional dialect list below as a native dialect. It is possible to understand someone who speaks a different dialect of the same language, but this requires a dialect check.

Much like competency in Skills, a character’s competency in a given language is determined by how many ranks he or she has taken in that language. The table below shows the varying levels of competency that character has in a given language. 


Dialects. Each language has several dialects, and every character will have a native dialect which matches his home region. If a character has achieved mastery of a language (5 ranks), he or she may then choose to master a local dialect of the language. Ranks of dialects are purchased with skill points exactly as ranks in languages. You must have 2 ranks in the base language to take 2 ranks of a dialect for that language.

1 Rank: The Character can speak and understand the dialect very well and quickly. Has a –2 penalty to Bluff, Diplomacy, Intimidate and Performance.

2 Ranks: The character is perfectly proficient with the language and can pass as a local. No penalties to any relevant skills.

Synergy: A character that has 5 or more ranks in Knowledge: Region for a given region will automatically count as having 1 rank in that region’s dialect. If the character already has a rank in that dialect, it counts as a having 2.   

Language Check: When communication is attempted between two characters who speak different dialects of the same language, a Dialect Check is required. This is an INT based check with a base DC of 10, which each character must pass. The DC is modified by the difficulty of the message being conveyed. For example, simple self-introductions may allow for a –2 modifier to the DC. In contrast, passing on the formula for a rare and magical potion, may require a DC modifier as high as +20. This check can be attempted three times for each message/idea. If it is failed after the third attempt, it is impossible for these two characters to understand each other for that particular missive.
