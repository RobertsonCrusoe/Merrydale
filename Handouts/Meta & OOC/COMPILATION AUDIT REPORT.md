FLAGGED ISSUES
================================================================================

1. DATE DISCREPANCIES IN FILENAMES:
   - "20 Ches 1484 DR - A Note from Godwin Ferguson.txt" — The PDF TOC also says
     "20 Ches, 1484 DR" so this is intentional (a note received during 1483 events
     but dated 1484, or a future-dated document). Not an error.
   - "Ches 21 1484 DR - Invitation from Mayor Cassimar.txt" — PDF says "21 Ches"
     with "Ches 21, 1483 DR" in the body. Filename should be 1483, not 1484.

2. DUPLICATE FILE VERSIONS:
   Several handouts have multiple versions (truncated filenames with " 2" suffix).
   These appear to be OCR-quality variants. A dedup pass would be worthwhile.

3. DAGGERFALLS GAZETTEER SUBSECTIONS:
   The PDF has a "Daggerfalls Gazetteer – Players' Handout" section (p.33–48)
   with subsections for Cathedral, Cassimar Tower, The Old Market, and The Myth
   Drannor Embassy. Only "Cassimar Tower.txt" exists as a separate file.
   The others may be embedded within "DDC 1482 3 Daggerdale Gazetteer.txt"
   or may warrant separate extraction.

4. OCR ARTIFACTS IN EXISTING FILES:
   The Tergus Clarandal journal files contain OCR artifacts (e.g., ".nd" for "find",

   ".rst" for "first", ".lled" for "filled"). The PDF compilation has cleaner text.
   A cleanup pass using the PDF as reference would improve these files.
