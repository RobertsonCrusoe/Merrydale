Here are a few links to sites which explain the basic markup syntax for editing the wiki. I will add an explanation of the templates that I've been using when I get a chance:

== Websites with simple editing advice and basic syntax ==

* [http://en.wikipedia.org/wiki/Wikipedia:How_to_edit_a_page Basic guide to editing a page]

* [http://en.wikipedia.org/wiki/Wikipedia:WikiProject_User_Page_Help/Do-It-Yourself/Formatting_Text Simple reference for formatting Text]

* [http://en.wikipedia.org/wiki/Wikipedia:WikiProject_User_Page_Help/Do-It-Yourself/Formatting_Pictures Simple reference for formatting Pictures]

* [http://people.fas.harvard.edu/~sdouglas/table.cgi Copy & Paste Excel-to-Wiki Converter]
** This tool is useful if you'd like to put in a large table of data without coding it yourself. Just make a table in Excel, copy and paste into the site, then paste the resulting code on the wiki.

* [http://toolserver.org/~diberri/cgi-bin/html2wiki/index.cgi HTML to Wiki Converter]
** A more advanced converter which also allows you to bring HTML formatting into your page.
