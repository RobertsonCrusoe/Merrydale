The Chrysalis Chronicles - Todd's Campaign Document Project
by Todd Burry
Email threads: "The Chrysalis Chronicles" (20-21 Dec 2023) and "Chrysalis Chronicles has been updated" (4-10 Feb 2024)

Todd Burry initiated a project to collect all campaign emails into a single collaborative Google Doc, formatted for tablet reading and potential print-on-demand. He wrote a session summary and invited others to contribute, hoping Stephen (the scribe) and others would add to the narrative. The document was updated after each session.

TODD (20 Dec 2023):

I've started collecting all the campaign emails we've been sending into a single document. My intention is for anyone to add to it if they would like. Otherwise, I just don't want to lose any of this great writing that has been going around.

I added a small summary of our session, but I know it could be better. I'm hoping that this is where our scribe can come in. Stephen might also want to weigh in to clean up Lucan's reputation.

A few other notes:

1. I have formatted the document on a smaller paper. I want this to be something that will work well on tablets. It is also a size that could be nicely printed using print-on-demand.
2. The mobile version of Google Docs does a pretty good job of rendering the document.
3. If you add to the doc, I can fix the formatting, but please try following these guidelines.
   - Use the paragraph styles for a consistent look.
   - If you are pasting in content, try using Ctrl-Shift-V to paste so that you don't paste formatting into the doc.
