Characters who are able to convince the Steward of the Embassy that they are in need of some gear might be able to acquire some goods from the armoury. This is a skill challenge.  

Skills:  Diplomacy, Bluff (see table below)

Failures:  3 failures ends the skill challenge, and closes off this resource for that character for a significant length of time.  Each failure adds 2 to the DC.

Successes:  As many as 6 successes can be achieved.  Any successes above 3 will provide a commerce option for the characters, but, like getting 3 failures, will close off this option for a significant length of time (Maxentius says \"That's about all the help we can provide for now, friend\").  Each success adds 1 to the DC (see the table, below)

Quitting:  This challenge can be stopped at any time, with any amount of success or failures.  A character may attempt this skill challenge again after a short amount of time (gaining a level, accomplishing a task for the embassy, etc).
